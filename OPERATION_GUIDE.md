# 跨境电商ERP系统操作指南

## 系统登录

1. 打开浏览器，访问系统地址（默认：http://localhost:3000）
2. 在登录页面输入账号密码：
   - 用户名：admin
   - 密码：admin123
3. 点击「登录」按钮进入系统

## 系统布局

系统采用经典的后台管理系统布局：
- **左侧菜单栏**：包含所有功能模块的导航
- **顶部导航栏**：显示面包屑导航和用户信息
- **主内容区**：显示当前页面的具体内容
- **页脚**：显示系统版本信息

## 核心功能操作指引

### 1. 运营中心 - 店铺管理

#### 功能说明
管理所有跨境电商平台的店铺，包括授权、状态管理等。

#### 操作步骤
1. 点击左侧菜单「运营中心」→「店铺管理」
2. **新增店铺**：点击「新增店铺」按钮，填写店铺信息并保存
3. **店铺授权**：选择店铺后点击「批量授权」按钮
4. **状态管理**：点击店铺操作列的「启用/禁用」按钮
5. **查看详情**：点击「查看」按钮查看店铺详细信息
6. **筛选搜索**：使用顶部筛选条件进行店铺搜索

### 2. 商品中心

#### 商品管理
1. 点击左侧菜单「商品中心」→「商品管理」
2. **新增商品**：点击「新增商品」按钮，填写商品信息
3. **批量操作**：选择商品后可进行批量上架、下架、删除、导出
4. **商品详情**：点击「查看」按钮查看商品详细信息，包括多语言信息、合规信息等
5. **状态管理**：点击「上架/下架」按钮切换商品状态

#### 分类管理
1. 点击左侧菜单「商品中心」→「分类管理」
2. **新增分类**：点击「新增分类」按钮，填写分类信息
3. **编辑分类**：点击「编辑」按钮修改分类信息
4. **状态管理**：点击「启用/禁用」按钮切换分类状态
5. **批量删除**：选择分类后点击「批量删除」按钮

### 3. 订单中心

#### 订单管理
1. 点击左侧菜单「订单中心」→「订单管理」
2. **订单筛选**：使用顶部筛选条件按平台、店铺、状态等筛选订单
3. **订单详情**：点击「查看」按钮查看订单详细信息
4. **订单操作**：点击「审核」、「发货」等按钮进行订单处理
5. **批量操作**：选择订单后可进行批量导出、审核等操作

#### 售后管理
1. 点击左侧菜单「订单中心」→「售后管理」
2. **售后处理**：点击「处理」按钮处理售后申请
3. **状态管理**：点击「审核」、「拒绝」等按钮更新售后状态
4. **退款管理**：处理退款申请并记录退款金额

### 4. 财税合规中心

#### 基础配置
1. 点击左侧菜单「财税合规中心」→「基础配置」
2. **汇率管理**：添加和管理不同币种之间的汇率
3. **税则配置**：配置全球各国家/地区的税率规则
4. **科目配置**：设置跨境业务的财务科目映射

#### VAT管理
1. 点击左侧菜单「财税合规中心」→「VAT管理」
2. **税号管理**：维护各站点的VAT/EORI税号
3. **税金核算**：查看订单维度的VAT计算
4. **申报管理**：生成和提交VAT申报单

#### 利润核算
1. 点击左侧菜单「财税合规中心」→「利润核算」
2. **多维度利润分析**：查看订单、商品、店铺等维度的利润
3. **费用分摊**：配置和查看费用分摊规则
4. **利润报表**：查看实时利润看板和分析报表

### 5. 数据报表

1. 点击左侧菜单「数据报表」→「经营看板」
2. 查看核心经营指标和趋势图表
3. 点击「销售报表」查看销售数据统计
4. 点击「库存报表」查看库存相关数据

### 6. 系统管理

1. 点击左侧菜单「系统管理」→「角色权限」
2. **角色管理**：创建和管理系统角色
3. **权限配置**：为角色分配菜单和按钮权限
4. 点击「用户管理」管理系统用户账号

## 系统快捷键

- **F5**：刷新当前页面
- **Ctrl + K**：快速搜索
- **Esc**：关闭当前弹窗或抽屉

## 常见问题

### 1. 店铺授权失败
- 检查网络连接
- 确认平台账号密码正确
- 检查平台API权限设置

### 2. 订单同步不及时
- 检查店铺授权状态
- 查看系统日志
- 手动触发同步

### 3. 财税数据不准确
- 检查汇率设置
- 确认税则配置正确
- 验证订单数据完整性

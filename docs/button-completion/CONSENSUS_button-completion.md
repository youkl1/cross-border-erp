# 按钮功能完善需求共识文档

## 1. 明确的需求描述

### 1.1 库存管理页面
- **新增库存**：添加新的库存记录，包括商品编码、商品名称、仓库、总库存、已预留库存等信息，支持表单验证和数据提交
- **批量调整**：对选中的库存进行批量数量调整，支持增加、减少、设置三种调整类型，用户可输入调整数量和调整原因
- **批量锁定**：将选中的库存批量设置为锁定状态，更新已预留和可用库存数量
- **批量解锁**：将选中的库存批量设置为解锁状态，更新已预留和可用库存数量
- **批量导出**：将选中的库存数据导出为JSON文件，文件名包含日期信息

### 1.2 售后管理页面
- **批量批准**：批量批准选中的售后申请，更新状态为处理中，并记录处理记录
- **批量拒绝**：批量拒绝选中的售后申请，更新状态为已拒绝，并记录处理记录
- **批量导出**：将选中的售后申请数据导出为JSON文件，文件名包含日期信息

- Vue Router for navigation
